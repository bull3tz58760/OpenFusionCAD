# OpenFusionCAD

Open-source alternative to Autodesk Fusion 360 built with Python.

## Setup Instructions

1. Create and activate a virtual environment:
```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. Install requirements:
```bash
pip install -r requirements.txt
```

3. Run the app:
```bash
python app/main.py
```
